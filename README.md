# Asteroids Game

A classic arcade-style Asteroids game built with HTML5 Canvas and JavaScript.

## Features
- Physics-based movement (thrust, friction).
- Screen wrapping (moving off one side brings you to the other).
- Projectile firing and asteroid splitting logic.
- Retro arcade aesthetic.

## How to Run
1. Clone the repository.
2. Open `index.html` in any modern web browser.

## Controls
- **Arrow Keys / WASD:** Movement and Thrust.
- **Spacebar:** Fire.
